二、YUM配置
1.进入yum文件夹
	cd /etc/yum.repos.d/
2.配置wget下载工具
	Yum -y install wget
3.下载阿里云的YUM
	wget  http://mirrors.aliyun.com/repo/Centos-7.repo
3.备份原repo文件
	Mv 旧名字 新名字
4.替换系统原repo文件
	Mv Centos-7.repo 新名字
5.执行yum源更新
	Yum clean all
	Yum makecache
Yum update

三、JDK配置
1.检索YUM源里面的JDK列表
	Yum list java-1.8*
2.执行JDK1.8的安装
	Yum install java-1.8.0-openjdk* -y
3.检查安装是否成功
Java -version
